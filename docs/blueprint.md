# **App Name**: ChronoFeed

## Core Features:

- Newsfeed Display: Display a scrollable newsfeed with sample posts including text, images, and like/comment/share controls.
- Top Navigation: Top navigation bar to switch sections, and for search.
- Sidebar Navigation: A collapsible left sidebar for quick navigation to main features.
- Post Creation: Simplified post creation input with text input field and basic formatting options.

## Style Guidelines:

- Primary color: Dark blue (#3B5998) mimicking Facebook's main brand color.
- Background color: Charcoal gray (#333333) to establish the dark mode base.
- Accent color: Subtle accent blue (#89a9d3) for interactive elements to provide focus.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, objective look; suitable for headlines or body text
- Use simplified, white/light gray icons for navigation and interactions (like, comment, share).
- Maintain consistent spacing and alignment for a clean, structured look in dark mode.
- Subtle transitions for interactions to maintain visual interest without being distracting.
